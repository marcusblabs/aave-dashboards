# Aave v4 Dashboards

Live, on-chain dashboards for Aave v4 and the DEX Spoke initiative. Each page reads
state directly from public RPC endpoints in the browser — no backend, no API keys.

- **[Tokenization Spoke Caps](https://marcusblabs.github.io/aave-dashboards/spoke-caps/)** —
  deposit-cap headroom on the `waCore*` ERC-4626 wrappers across the Ethereum and
  Avalanche Core Hubs (cap, supplied, available per wrapper).
- **Fluid DEX Vaults** — market analysis for the Aave DEX Spoke project (in progress).

Unofficial community dashboards, not affiliated with Aave. Nothing here is financial advice.
